This directory contains instructions you need to follow to implement next iterations of kata.
Please go through them sequentially and go to next one only after completing previous one!

For the scope of this kata, the player won't be running around and doing stuff. The features implemented in each iteration will be limited to the framework that the game may use in the future.
It is centered on the characters combat skills which are basically damaging and healing ... but if you want you can extend it in any way just for fun after finishing the last iteration! For sure it will be appreciated during retrospective.
You may fix a time for each iteration as a constraint - if you want.
